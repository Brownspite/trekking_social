# Trekking Social - Project Description

**Project Goal**: Build a web application for a community focused mainly on trekking and outdoor activities, but that also organizes social events such as dinners, meetups and group experiences. The goal is to create a simple and clean platform where users can discover events and easily join them.

## Technical Approach: Web App & PWA
- **Web App**: Preferred over a traditional website because the platform should feel like a community tool, not just a static page.
- **PWA (Progressive Web App)**: Users must be able to add it to their phone's home screen and use it like a lightweight mobile app.
- **Mobile First**: Most users will access the platform from their phone, making the mobile user experience extremely important.

## Core Features

### 1) Event System
- Event calendar
- Event page with details (location, description, difficulty for trekking events)
- Join/register for events
- Limit number of participants
- Participant list

### 2) User Accounts
- User registration / login
- Basic user profile
- Simple dashboard showing upcoming events


## UI / Design Requirements
- Look modern and clean
- Feel simple and intuitive
- Work perfectly on mobile

## Deliverables Expected
- Fully working web application
- Clean mobile-first UI
- PWA capability
- Basic deployment instructions
